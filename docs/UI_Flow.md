**瓶子区域布局规则**:
- **3-4个瓶子**: 单行横向居中排列
- **5-6个瓶子**: 两行排列（上3下3或上2下4）
- **7-8个瓶子**: 两行排列（上4下4）
- **9-10个瓶子**: 两行或三行自适应
- 瓶子间距: 固定值，根据屏幕宽度动态缩放瓶子大小

**选中状态表现**:
- 选中的瓶子 **向上浮动10px**
- 瓶身添加 **金色外发光**（Glow效果）
- 瓶底出现 **圆形阴影**
- 播放选中音效

## 3. 适配方案

### 3.1 屏幕比例适配

| 比例 | 典型机型 | 适配策略 |
|------|----------|----------|
| 16:9 | iPhone 8 | 标准布局，左右留边 |
| 19.5:9 | iPhone 14 | 瓶子区域适当压缩，UI向安全区靠拢 |
| 20:9 | 安卓全面屏 | 背景图扩展，核心操作区保持居中 |
| 平板 | iPad | 瓶子区域放大，间距增加，充分利用大屏 |

### 3.2 微信安全区适配

```typescript
// 获取安全区
const systemInfo = wx.getSystemInfoSync();
const safeArea = systemInfo.safeArea;

// 顶部安全距离（刘海屏）
const topSafeHeight = safeArea.top;

// 底部安全距离（iPhone X+ 底部横条）
const bottomSafeHeight = systemInfo.screenHeight - safeArea.bottom;

// UI节点适配
uiNode.getComponent(Widget).top = topSafeHeight;
uiNode.getComponent(Widget).bottom = bottomSafeHeight;
```

### 3.3 字体适配
- 使用 `cc.Label` 的 `fontSize` 根据屏幕宽度等比缩放
- 基准: 750px宽屏幕下，标题字48px，正文32px，按钮文字36px
- 大屏设备最大放大到1.2倍，避免字体过大

## 4. 动效规范

### 4.1 通用动效参数

| 动效类型 | 时长 | 缓动函数 | 说明 |
|----------|------|----------|------|
| 界面弹出 | 0.3s | BackOut | 带轻微回弹 |
| 界面关闭 | 0.2s | QuadIn | 快速收起 |
| 按钮点击 | 0.1s | QuadOut | 缩放至0.9再恢复 |
| 物品出现 | 0.4s | BackOut | 缩放+淡入 |
| 数字滚动 | 0.5s | QuadOut | 逐帧插值 |

### 4.2 游戏专属动效

| 动效 | 时长 | 说明 |
|------|------|------|
| 倒水动画 | 0.5s | 液体流动+填充同步 |
| 选中浮动 | 循环 | 上下浮动5px，2s周期 |
| 错误抖动 | 0.3s | 左右快速抖动3次 |
| 通关庆祝 | 2.0s | 星星飞出+彩纸粒子 |
| 撤销倒流 | 0.4s | 倒放感，快速回溯 |

### 4.3 粒子特效

| 特效 | 触发时机 | 参数 |
|------|----------|------|
| 倒水水花 | 液体落入目标瓶瞬间 | 10个粒子，白色，向上飞溅，存活0.3s |
| 通关彩纸 | 结算界面弹出 | 30个彩色纸片，重力下落，存活2.0s（v0.2基础版）|
| 获得星星 | 星星评价显示时 | 星星节点放大+发光，周围散发6个光点 |
| 开关切换 | 设置面板开关点击 | 圆点滑动轨迹残影（可选）|
