# 《倒水大师》v0.3 功能需求文档（草案）

> **文档版本**: v0.1（草案）  
> **日期**: 2026-04-30  
> **对应里程碑**: v0.3  
> **目标**: 在 v0.2 单局体验版基础上，补齐“可发布到微信测试”的关键闭环能力：微信登录、云存档同步、分享（文案与奖励发放）、好友排行榜（开放数据域基础版）。

---

## 0. 范围声明

### 0.1 v0.3 计划包含（In Scope）

- **微信登录**：拉起 `wx.login`，拿到 code，具备后端/云函数对接的接口预留。
- **存档云同步**：本地存档与云存档双向同步（至少：拉取+覆盖策略），具备冲突解决规则。
- **分享功能 v1**：内置文案库 + 关键场景触发 + 奖励发放（与经济配置对齐）。
- **好友排行榜 v1**：开放数据域展示好友排行（以星数/通关数为指标之一）。

### 0.2 v0.3 不包含（Out of Scope，后续版本）

- 激励视频/插屏/Banner 广告接入（规划 v0.4）
- 去广告内购（规划 v0.4）
- 主题商店、每日挑战、成就正式版（规划 v0.5）
- 关卡热更新/分包（可后置）

## 1. 微信登录（Login）

### 1.1 需求

- 进入主菜单后，若未登录，则以“静默优先”策略发起登录（不强制授权头像昵称）。
- 登录失败不阻塞试玩，但需有可见提示与重试入口。

### 1.2 验收标准

- [ ] 主菜单启动时可触发 `wx.login` 并获得 code（可在日志中看到）。
- [ ] 登录失败会 Toast 提示且可重试（不崩溃、不死循环）。
- [ ] 登录状态会写入本地存档（或 session 内缓存），下次启动可复用。

---

## 2. 云存档同步（Cloud Save Sync）

### 2.1 数据范围

以 `PlayerSaveData` 为主（关卡进度、星级、金币/钻石、设置、签到、统计等）。

### 2.2 同步策略（建议）

- **启动拉取**：当天首次启动/登录成功后拉取云存；若云存不存在则以本地为准并上报。
- **关键时刻上报**：通关结算/签到/设置变更/重置存档等操作后，触发云存写入（可做节流）。
- **冲突解决**：以 `lastSaveTime` 时间戳较新的为准；对 `currentLevel`/`levelStars` 做“取大值/合并”。

### 2.3 验收标准

- [ ] 可从云端拉取存档并覆盖到本地（可通过 GM 面板摘要验证数据变化）。
- [ ] 通关一关后可把进度写入云端（至少日志可见）。
- [ ] 同一账号在两次启动间能恢复进度（“云→本地”生效）。

---

## 3. 分享功能 v1（Share + Rewards）

### 3.1 分享场景

- **主菜单分享求助**：分享一次给 50 金币（每天首次）。
- **关卡卡关求助**：弹窗触发分享，奖励提示次数或金币（按经济配置）。
- **通关炫耀**：结算界面分享，不强制奖励（可选）。

### 3.2 文案库

提供多条随机文案模板（含变量：关卡、步数、星级等），避免重复。

### 3.3 奖励发放

- 与 `Economy.md` / `GameConfig.ts` 对齐：奖励写入本地存档，并在 UI 上即时刷新。
- 防刷：每日次数限制（可先做本地限制，后续接服务端防刷）。

### 3.4 验收标准

- [ ] 能在微信环境中调用分享 API（或最小可用占位流程）。
- [ ] 分享成功后奖励正确写入存档，且 UI 可见更新。
- [ ] 每日首次/次数限制规则正确生效。

---

## 4. 好友排行榜 v1（Open Data Context）

### 4.1 指标

优先：**总星数**、其次：**最高关卡**、**通关数**。

### 4.2 展示与入口

- 主菜单“排行”入口进入排行榜面板（v0.2 为占位，v0.3 替换为可用面板）。
- 排行榜面板可切换：好友排行 / 周排行（周排行可先占位）。

### 4.3 验收标准

- [ ] 能正常打开开放数据域并显示至少 1 条好友排行数据（无好友则显示空态）。
- [ ] 主域向开放数据域发送消息、开放数据域响应绘制流程稳定（无报错）。
- [ ] 退出排行榜能回到主菜单。

---

## 5. v0.3 验收用例（最小回归清单）

- [ ] 启动 → 登录 → 拉取云存 → 主菜单
- [ ] 通关一关 → 写云存 → 重启 → 进度保持
- [ ] 主菜单分享 → 发放奖励 → 次数限制
- [ ] 打开排行榜 → 展示 → 返回

---

## 6. 开发实现记录（2026-04-30）

- **微信登录**：已由 `SDKManager.loginAndSync` 启动静默登录；编辑器环境返回 mock code，微信环境调用 `wx.login`。
- **云存档同步**：已新增 `CloudSaveManager`，优先预留 `wx.cloud.callFunction(saveData/loadData)`，编辑器环境使用本地 shadow 存档降级；通关、签到、设置变更、重置存档后会上报。
- **分享功能 v1**：已新增 `ShareManager` 文案库；主菜单“分享求助”每日首次奖励 50 金币，结算页支持分享炫耀。
- **好友排行榜 v1**：已新增 `RankManager`；主菜单“排行”进入排行榜面板，微信环境会向开放数据域发送消息，编辑器环境展示预览数据。
- **待真机确认**：开放数据域实际绘制、云函数 `saveData/loadData`、微信分享回调限制需在微信开发者工具/真机环境补测。
