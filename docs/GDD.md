## 1. 游戏概述

### 1.1 游戏名称
- **主推荐**: 《倒水大师》
- **备选**: 《色彩分水》、《水杯排序》、《Liquid Sort》

### 1.2 一句话描述
> 将混杂在不同水杯中的彩色液体，通过倾倒分类，让每个杯子只装同一种颜色——看似简单，越往后越烧脑！

### 1.3 Elevator Pitch
《倒水大师》是一款轻量级颜色分类益智游戏。玩家通过点击操作将不同颜色的液体在玻璃杯间倾倒，目标是将同种颜色归类到同一个杯子中。游戏采用清新简约的视觉风格，单局1-3分钟，适合碎片化时间游玩。内置500+精心设计的关卡，集成微信分享、排行榜和激励视频广告，具备完整的IAA变现体系。

### 1.4 核心循环 (Core Loop)

```
选择关卡 → 观察颜色分布 → 点击选瓶 → 点击目标瓶（倒水）→ 判断是否完成
    ↑                                                    ↓
    └──────── 通关奖励 ← 领取金币/星星 ← 结算界面 ───────┘
              ↓
        看广告获得双倍奖励 / 分享解锁新关卡
              ↓
        进入下一关 / 返回选关界面
```

### 1.5 目标用户画像

| 维度 | 描述 |
|------|------|
| **核心年龄** | 18-45岁 |
| **性别倾向** | 女性略多（55%），男性45% |
| **场景** | 通勤、午休、睡前、排队等候 |
| **心理需求** | 解压、轻度烧脑、收集通关成就感、好友间比拼 |
| **付费意愿** | 低-中等，更愿意看广告换取便利 |

## 2. 核心玩法设计

### 2.1 基础规则

1. **同色叠加**: 只有当目标瓶的最顶层颜色与源瓶最顶层颜色**相同**，或目标瓶为**空瓶**时，才能倾倒。
2. **连续倾倒**: 一次点击会将源瓶中**连续相同的所有顶层颜色**一次性倒入目标瓶（例如源瓶顶部有2格红色，则2格同时倒入）。
3. **容量限制**: 每个瓶子有固定的最大容量（统一为4格）。目标瓶剩余空间不足以容纳待倾倒的液体时，无法倾倒。
4. **空瓶策略**: 空瓶可作为"缓冲区"暂存任意颜色，是解谜的关键策略点。

### 2.2 操作方式

| 步骤 | 玩家操作 | 系统反馈 |
|------|----------|----------|
| 1 | 点击一个非空瓶子 | 瓶子高亮（选中状态），播放"叮"音效 |
| 2 | 点击另一个有效目标瓶 | 播放倒水动画，液体从源瓶"流"入目标瓶 |
| 3 | 点击无效目标瓶 | 播放错误音效，目标瓶轻微抖动（拒绝反馈） |
| 4 | 点击已选中的瓶子 | 取消选中状态 |

**操作优化**:
- 支持"快速双击": 双击一个瓶子，系统自动寻找唯一可行目标并执行（降低新手门槛）
- 支持"滑动选瓶": 在瓶子上滑动可以快速切换选中状态（可选高级功能）

### 2.3 胜利条件
- 所有非空瓶子内的液体颜色**完全一致**
- 空瓶可以存在（不要求全部填满）

### 2.4 失败/重置机制
本游戏**没有传统意义上的失败**（无时间限制、无步数限制），但提供以下功能：

| 功能 | 说明 | 消耗 |
|------|------|------|
| **重新开始** | 重置当前关卡到初始状态 | 免费 |
| **撤销 (Undo)** | 回退一步操作 | 免费前3次，后续看广告或消耗金币 |
| **提示 (Hint)** | 高亮推荐的一步操作 | 看激励视频 或 消耗50金币 |

### 2.5 撤销功能设计
- **栈结构记录**: 使用操作栈记录每一步的（源瓶索引、目标瓶索引、倾倒层数）
- **免费额度**: 每关前3次撤销免费（培养使用习惯）
- **超额收费**: 第4次起，每次撤销消耗10金币，或看15秒激励视频获得5次免费撤销
- **视觉表现**: 撤销时播放"倒流"动画（与正向倒水动画方向相反，时长减半）

## 3. 关卡设计系统

### 3.1 难度参数

关卡难度由以下参数控制：

| 参数 | 说明 | 初期值 | 中期值 | 后期值 |
|------|------|--------|--------|--------|
| **颜色种类** | 不同颜色的数量 | 2-3 | 4-5 | 6-8 |
| **瓶子总数** | 场上杯子数量（含空瓶） | 4-5 | 6-7 | 8-10 |
| **空瓶数量** | 纯空瓶的数量 | 1 | 1-2 | 1-3 |
| **每层高度** | 每瓶初始填充高度 | 2-3 | 3-4 | 4（满） |
| **干扰色** | 故意放置的误导色块 | 无 | 少量 | 较多 |

### 3.2 难度曲线

- **第1-10关**: 教学关，2-3种颜色，明显解法，让玩家熟悉规则
- **第11-50关**: 入门阶段，3-4种颜色，需要1-2步预判
- **第51-200关**: 进阶阶段，4-5种颜色，需要利用空瓶做中转
- **第201-500关**: 挑战阶段，5-6种颜色，多层嵌套逻辑
- **第500关+**: 大师阶段，6-8种颜色，极限烧脑，空瓶极少

### 3.3 关卡可解性保证 (Solvability Check)

所有关卡在发布前必须通过算法验证**存在可行解**：

```
算法思路（BFS/DFS）:
1. 从初始状态出发
2. 枚举所有合法倾倒操作
3. 检查是否到达胜利状态
4. 记录最少步数作为"三星基准"
```

- 每个关卡记录 **最少步数** 和 **建议步数**
- 玩家用步数 ≤ 最少步数+2 → 3星
- 玩家用步数 ≤ 最少步数+5 → 2星
- 玩家用步数 > 最少步数+5 → 1星

## 4. 微信小游戏特性集成

### 4.1 分享功能

| 分享场景 | 触发时机 | 分享文案 | 奖励 |
|----------|----------|----------|------|
| **每日首次分享** | 主界面分享按钮 | "我在倒水大师第{level}关卡住了，求助大神！" | 50金币 |
| **关卡求助** | 卡关时弹窗 | "这关太难了，求好友指点迷津！" | 获得3次免费提示 |
| **通关炫耀** | 通关结算页 | "我只用{steps}步就通关了第{level}关，来挑战我！" | 无直接奖励，社交传播 |
| **排行榜分享** | 排行榜界面 | "我在倒水大师排行榜排名第{rank}，不服来战！" | 无 |

**分享图规则**: 使用动态Canvas绘制，包含玩家头像（如授权）、当前关卡截图、游戏二维码。

### 4.2 广告系统

#### 激励视频广告 (Rewarded Video)
- **触发点1**: 提示下一步 → 看完获得1次提示
- **触发点2**: 撤销步数用完 → 看完获得5次免费撤销
- **触发点3**: 关卡结算 → 看完获得双倍金币奖励
- **触发点4**: 解锁新主题 → 看完直接解锁（替代金币支付）
- **触发点5**: 每日签到漏签 → 看完补签

**广告冷却**: 同一触发点间隔至少30秒；每日最多展示20次激励视频。

#### 插屏广告 (Interstitial)
- **展示时机**: 每完成3关后，在选关界面展示
- **频次控制**: 每5分钟最多1次；用户首次游玩前30分钟不展示

#### Banner广告
- **展示位置**: 屏幕底部（避开操作区）
- **展示时机**: 游戏内界面常驻；主菜单、选关界面、结算界面显示
- **隐藏时机**: 实际倒水操作过程中短暂隐藏（避免误触）

### 4.3 排行榜

- **好友排行榜**: 基于开放数据域，显示微信好友的通关进度和总星数
- **周排行榜**: 每周一0点重置，按本周获得星数排名
- **全球排行榜**: 按总通关数排名（分省/市显示，增加地域竞争感）

### 4.4 订阅消息

| 消息类型 | 触发条件 | 文案模板 |
|----------|----------|----------|
| 每日签到提醒 | 用户24小时未登录 | "您已连续签到{days}天，今日奖励翻倍，快来领取！" |
| 新关卡解锁 | 章节更新时 | "第{chapter}章全新关卡已上线，快来挑战！" |
| 好友超越 | 被好友在排行榜超越 | "好友{friendName}刚刚超越了你，快去夺回排名！" |

## 5. 经济系统与商业化

### 5.1 双货币体系

| 货币 | 获取方式 | 用途 | 产出控制 |
|------|----------|------|----------|
| **金币** | 通关奖励、每日签到、看广告、分享 | 购买提示、购买撤销次数、解锁主题 | 大量产出，轻度通货膨胀可接受 |
| **钻石** | 内购、特殊活动、成就奖励 | 去广告包、限定主题、大额金币兑换 | 严格控制，主要付费点 |

### 5.2 收入来源

| 收入类型 | 预计占比 | 说明 |
|----------|----------|------|
| **激励视频广告** | 60% | 核心收入来源，用户主动选择观看 |
| **插屏广告** | 20% | 被动展示，控制频次保体验 |
| **Banner广告** | 10% | 常驻底部，稳定收入 |
| **内购** | 10% | 去广告包(￥6)、钻石礼包、主题包 |

### 5.3 内购商品设计

| 商品 | 价格 | 内容 | 目标用户 |
|------|------|------|----------|
| **去广告特权** | ￥6 | 永久移除插屏和Banner广告，保留激励视频（因激励是玩家主动行为） | 中高频玩家 |
| **新手礼包** | ￥1 | 500金币 + 50钻石 + 3个提示道具 | 新用户首充 |
| **大师礼包** | ￥12 | 2000金币 + 200钻石 + 解锁全部主题 | 核心玩家 |
| **钻石月卡** | ￥18 | 每日领取30钻石，持续30天 | 长期留存用户 |

## 7. 音效与音乐设计

### 7.1 BGM
- **主菜单**: 轻快、舒缓的钢琴曲，循环播放
- **游戏内**: 极简环境音（水滴声、微风声），不干扰思考
- **结算**: 简短的胜利和弦

### 7.2 音效清单

| 事件 | 音效风格 | 时长 |
|------|----------|------|
| 选中瓶子 | 清脆玻璃声 "叮" | <0.5s |
| 倒水开始 | 水流倾倒声 | 配合动画时长 |
| 倒水结束 | 水滴落入声 "咚" | <0.3s |
| 错误操作 | 闷声 "噗" + 轻微震动 | <0.3s |
| 撤销 | 时间倒流音效（倒放感） | <0.5s |
| 通关 | 连续清脆音符上升 | 2-3s |
| 获得星星 | "叮叮叮" 连续三声 | 1s |
| 按钮点击 | 通用UI点击音 | <0.2s |
