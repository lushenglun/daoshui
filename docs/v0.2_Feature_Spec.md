### 2.4 设置项详细行为

| 设置项 | 默认值 | 开启效果 | 关闭效果 | 持久化 |

| 音乐 | true | `AudioManager.setBGMVolume(0.5)` | `AudioManager.setBGMVolume(0)` | 立即写入存档 |

**注意**：音乐和音效开关关闭时，不应停止正在播放的音频，而是将音量设为 0。重新开启时恢复原始音量。

## 5. GameState 状态机更新

v0.2 新增 `PAUSED` 状态，需要更新状态流转：

```typescript
export enum GameState {
    LOADING = 'LOADING',
    MAIN_MENU = 'MAIN_MENU',
    LEVEL_SELECT = 'LEVEL_SELECT',
    PLAYING = 'PLAYING',
    PAUSED = 'PAUSED',           // ← v0.2 新增
    LEVEL_COMPLETE = 'LEVEL_COMPLETE',
    SETTINGS = 'SETTINGS',       // ← v0.2 新增（可从 PAUSED 或 MAIN_MENU 进入）
}
```

**状态流转约束**:
- `PLAYING` → `PAUSED`: 允许（暂停按钮、返回键、切后台）
- `PAUSED` → `PLAYING`: 允许（继续游戏）
- `PAUSED` → `MAIN_MENU`: 允许（返回主菜单，需二次确认）
- `PAUSED` → `SETTINGS`: 允许（设置按钮）
- `SETTINGS` → `PAUSED`: 允许（设置面板关闭）
- `SETTINGS` → `MAIN_MENU`: 允许（设置面板关闭）
- 任何状态 → `LOADING`: 允许（切换关卡时）

**输入屏蔽规则**:
- `PAUSED` 状态下：瓶子点击无效、撤销/提示/重置按钮无效
- `PAUSED` 状态下：仅响应弹窗内按钮和遮罩点击
- `LEVEL_COMPLETE` 状态下：瓶子点击无效，但结算弹窗按钮可用
