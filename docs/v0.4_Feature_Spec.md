## 1. 广告系统架构

### 1.1 模块关系

```
GameManager / UI Layer
    │
    ├──▶ AdManager（统一广告入口）
    │       │
    │       ├──▶ WXAPI.showRewardedVideo()
    │       ├──▶ WXAPI.showInterstitial()
    │       └──▶ WXAPI.createBanner() / showBanner() / hideBanner()
    │
    ├──▶ PlayerSaveData.adStats（广告次数统计）
    │

```

### 1.2 核心原则

1. **所有广告调用必须经过 AdManager**，不允许 UI 层直接调用 `wx` 广告 API
2. **广告加载失败必须有降级**，不能卡死玩家流程
3. **广告统计必须精确**，用于上限控制和风控

## 2. AdManager 完整实现

### 2.1 类定义

```typescript
export class AdManager {
    // === 激励视频 ===
    static preloadRewardedVideo(): void;
    static async showRewardedVideo(scene: RewardedScene): Promise<RewardedResult>;

    // === 插屏广告 ===
    static preloadInterstitial(): void;
    static async showInterstitial(): Promise<boolean>;

    // === Banner 广告 ===
    static createBanner(): void;
    static showBanner(): void;
    static hideBanner(): void;
    static destroyBanner(): void;

    // === 每日统计 ===
    static getAdStats(): AdStats;
    static resetDailyStatsIfNeeded(): void;
}

export type RewardedScene = 'hint' | 'undo' | 'double_coins' | 'free_coins';

export interface RewardedResult {
    success: boolean;      // 视频是否完整观看
    completed: boolean;    // 玩家是否看完（非跳过/关闭）
    message: string;       // 提示文案
}

export interface AdStats {
    rewardedToday: number;         // 今日已观看激励视频次数
    interstitialToday: number;     // 今日已展示插屏次数
    lastRewardedTime: number;      // 上次观看激励视频时间戳
    lastInterstitialTime: number;  // 上次展示插屏时间戳
}
```

### 2.2 激励视频实现细节

**预加载策略**：

| 场景 | 预加载时机 | 说明 |
|------|-----------|------|
| `hint` | 进入关卡时 | 玩家可能随时需要提示 |
| `undo` | 撤销用完时 | 按需加载，避免浪费 |
| `double_coins` | 通关结算时 | 结算弹窗弹出即预加载 |
| `free_coins` | 主菜单显示时 | 常驻预加载 |

**微信 API 封装**：

```typescript
// WXAPI.ts 新增
static async showRewardedVideo(adUnitId: string): Promise<boolean> {
    if (!this.available || !wx.createRewardedVideoAd) {
        return false;
    }

    return new Promise((resolve) => {
        const rewardedAd = wx.createRewardedVideoAd({ adUnitId });
        
        rewardedAd.onLoad(() => {
            rewardedAd.show().catch(() => resolve(false));
        });

        rewardedAd.onClose((res: { isEnded: boolean }) => {
            resolve(res.isEnded); // true = 完整观看
        });

        rewardedAd.onError((err: unknown) => {
            console.warn('[WXAPI] rewarded video error:', err);
            resolve(false);
        });

        rewardedAd.load();
    });
}
```

**关键行为**：
- 如果激励视频加载失败，返回 `{ success: false, completed: false, message: '广告加载失败，请稍后重试' }`
- 如果玩家中途关闭（未看完），返回 `{ success: true, completed: false, message: '观看完整视频才能获得奖励' }`
- 只有 `completed === true` 时才发放奖励
- 每次展示前检查 `AdStats.rewardedToday < GAME_CONFIG.AD.REWARDED_DAILY_LIMIT`
- 每次展示前检查 `Date.now() - lastRewardedTime > GAME_CONFIG.AD.REWARDED_COOLDOWN * 1000`

### 2.3 插屏广告实现细节

**展示策略**（引用 GameConfig）：

| 条件 | 配置值 |
|------|--------|
| 首次展示关卡 | `INTERSTITIAL_FIRST_LEVEL = 6` |
| 关卡间隔 | `INTERSTITIAL_LEVEL_INTERVAL = 3` |
| 冷却时间 | `INTERSTITIAL_COOLDOWN = 300` 秒 |
| 每日上限 | `INTERSTITIAL_DAILY_LIMIT = 20` |
| 展示时机 | 结算界面 → 选关界面 / 下一关 过渡时 |

**触发逻辑**：

```typescript
// 在 completeLevel 或返回选关/下一关时触发
function shouldShowInterstitial(levelId: number): boolean {
    // 纯广告模式，不检查去广告状态
    if (levelId < GAME_CONFIG.AD.INTERSTITIAL_FIRST_LEVEL) return false;
    if (levelId % GAME_CONFIG.AD.INTERSTITIAL_LEVEL_INTERVAL !== 0) return false;
    
    const stats = AdManager.getAdStats();
    if (stats.interstitialToday >= GAME_CONFIG.AD.INTERSTITIAL_DAILY_LIMIT) return false;
    if (Date.now() - stats.lastInterstitialTime < GAME_CONFIG.AD.INTERSTITIAL_COOLDOWN * 1000) return false;
    
    return true;
}
```

**微信 API 封装**：

```typescript
// WXAPI.ts 新增
static async showInterstitial(adUnitId: string): Promise<boolean> {
    if (!this.available || !wx.createInterstitialAd) {
        return false;
    }

    return new Promise((resolve) => {
        const interstitialAd = wx.createInterstitialAd({ adUnitId });

        });

        interstitialAd.onError((err: unknown) => {
            console.warn('[WXAPI] interstitial error:', err);

        });

```

### 2.4 Banner 广告实现细节

**生命周期管理**：

| 界面 | Banner 状态 | 说明 |
|------|-------------|------|
| 主菜单 (MAIN_MENU) | 展示 | 常驻底部 |
| 选关界面 (LEVEL_SELECT) | 展示 | 常驻底部 |
| 游戏界面 (PLAYING) | 隐藏 | 避免遮挡操作 |
| 暂停弹窗 (PAUSED) | 隐藏 | 弹窗期间不展示 |
| 结算界面 (LEVEL_COMPLETE) | 展示 | 常驻底部 |

**微信 API 封装**：

```typescript
// WXAPI.ts 新增
private static bannerAd: any = null;

static createBanner(adUnitId: string): void {
    if (!this.available || !wx.createBannerAd) return;
    
    this.destroyBanner();
    this.bannerAd = wx.createBannerAd({
        adUnitId,
        style: { left: 0, top: 0, width: 320 }, // 微信会自动适配
    });
}

static showBanner(): void {
    if (this.bannerAd) {
        this.bannerAd.show().catch((err: unknown) => console.warn('[WXAPI] banner show failed:', err));
    }
}

static hideBanner(): void {
    if (this.bannerAd) {
        this.bannerAd.hide();
    }
}

static destroyBanner(): void {
    if (this.bannerAd) {
        this.bannerAd.destroy();
        this.bannerAd = null;
    }
}
```

**状态切换时序**：

```typescript
// 在 GameManager 状态切换时调用
switch (newState) {
    case GameState.MAIN_MENU:
    case GameState.LEVEL_SELECT:
    case GameState.LEVEL_COMPLETE:
        AdManager.showBanner();
        break;
    case GameState.PLAYING:
    case GameState.PAUSED:
        AdManager.hideBanner();
        break;
}
```

## 3. 激励视频 4 个点位详细设计

### 3.1 点位1：提示按钮 (hint)

**触发条件**：玩家点击"提示"按钮时，金币不足 50 且当日免费提示已用完

**交互流程**：

```
玩家点击"提示"
    │
    ▼
检查是否有免费提示额度（v0.5实现，v0.4暂用金币逻辑）
    │
    ├── 有 ──▶ 直接显示提示
    │
    └── 无 ──▶ 检查金币 >= 50
                  │
                  ├── 是 ──▶ 消耗50金币，显示提示
                  │
                  └── 否 ──▶ 弹出选项：
                                "金币不足，观看视频免费获得1次提示？"
                                [取消] [看视频]
                                      │
                                      ▼
                                调用 AdManager.showRewardedVideo('hint')
                                      │
                                      ├── completed === true
                                      │       │
                                      │       ▼
                                      │   显示提示，记录今日已获得
                                      │
                                      └── completed === false
                                              │
                                              ▼
                                          Toast: "观看完整视频才能获得提示"
```

**UI 文案**：
- 弹窗标题："金币不足"
- 弹窗内容："观看15秒视频，免费获得1次提示"
- 按钮："取消" / "看视频"

### 3.2 点位2：撤销按钮 (undo)

**触发条件**：玩家点击"撤销"按钮时，已超出免费额度（3次）且金币不足 10

**交互流程**：

```
玩家点击"撤销"
    │
    ▼
检查 undoStack 是否为空
    │
    ├── 是 ──▶ Toast: "没有可撤销的步骤"
    │
    └── 否 ──▶ 检查今日免费撤销次数 < 3
                  │
                  ├── 是 ──▶ 免费撤销
                  │
                  └── 否 ──▶ 检查金币 >= 10
                                │
                                ├── 是 ──▶ 消耗10金币，撤销
                                │
                                └── 否 ──▶ 弹出选项：
                                              "免费撤销额度已用完，观看视频获得5次免费撤销？"
                                              [取消] [看视频]
                                                    │
                                                    ▼
                                              调用 AdManager.showRewardedVideo('undo')
```

### 3.3 点位3：双倍金币 (double_coins)

**触发条件**：结算界面点击"双倍奖励"按钮

**交互流程**：

```
结算界面显示
    │
    ▼
预加载激励视频（ AdManager.preloadRewardedVideo() ）
    │
    ▼
玩家点击"双倍奖励"
    │

检查今日双倍次数 < GAME_CONFIG.AD.REWARDED_DAILY_LIMIT
    │

    │
    └── 是 ──▶ 检查冷却时间
                  │
                  ├── 冷却中 ──▶ Toast: "请稍后再试"
                  │
                  └── 可展示 ──▶ 播放激励视频
                                      │
                                      ├── completed === true
                                      │       │
                                      │       ▼
                                      │   金币 ×2，播放获得动画
                                      │   按钮变为"已获得双倍"
                                      │
                                      └── completed === false
                                              │
                                              ▼
                                          Toast: "观看完整视频才能获得双倍奖励"
```

**按钮状态机**：

| 状态 | 视觉 | 交互 |

```typescript
// StorageManager 或 AdManager 中实现
static resetDailyStatsIfNeeded(): void {
    const data = this.load();
    const today = WXAPI.getLocalDateKey();
    
    if (data.adStats.lastResetDate !== today) {
        data.adStats.rewardedToday = 0;
        data.adStats.interstitialToday = 0;
        data.adStats.lastResetDate = today;
        this.save(data);
    }
}
```

## 5. 广告加载失败降级方案

### 6.1 降级原则

| 广告类型 | 加载失败时行为 |
|----------|---------------|
| 激励视频 | 提示"广告加载失败"，提供金币消耗替代方案（如果适用）|
| 插屏 | 静默跳过，不影响玩家流程 |
| Banner | 静默跳过，不影响玩家流程 |

### 6.2 激励视频降级详情

```typescript
// 在 AdManager.showRewardedVideo 中

        completed: false,
        message: '广告加载失败，请检查网络后重试',
    };
```

**UI 层处理**：
- `hint` 点位：广告失败时，如果金币足够，提示"是否消耗50金币购买提示？"
